Procedure to create and View a Webstite on the Production server

  1. Clone your repository (git clone).
  2. Change into your directory (cd "directory name".
  3. Create a new html file in your repository (gedit "file name".)
  4. Type the necessary codes to create a portal to your website in your new html file.
  5. Validate your html file.
  6. Create a new css file in your repository (gedit "file name").
  7. Edit your css file by adding declarations and properties.
  8. Validate your css file.
  9. Git add your files (git add "file name").
  10. Git commit (git commit -m "Commit Comment").
  11. Git push (git push).
  12. Use the sftp command to access the production server (sftp username@cmis-hsteacher)
  13. Change the directory in the production server to public_html (cd public_html).
  14. Put your files in the production server (put "file name").
